# Contributing to Docker Docs

We welcome contributions!

## Issues

If you find a bug or have a feature request, please [open an issue](https://github.com/northcutted/docker-docs/issues/new).

## Pull Requests

1. Fork the repository.
2. Create a new branch: `git checkout -b feature/my-feature`
3. Make your changes and commit them: `git commit -m 'feat: add my feature'`
4. Push to the branch: `git push origin feature/my-feature`
5. Submit a pull request.

## Development

1. Ensure Go is installed (>= 1.21).
2. Install dependencies: `go mod download`
3. Run tests: `go test ./...`

## Code Style

Follow standard Go conventions and use `gofmt` or `goimports` to format your code.
